# IPL Payment Update - Pembayaran IPL + Iuran Kegiatan dalam Satu Transfer

**Date:** 2025-01-14
**Version:** 1.0.0
**Author:** Claude Code

## Ringkasan

Update sistem pembayaran IPL untuk mendukung:
1. **Auto-generate Reference Number** - Tidak perlu input manual dari bukti transfer
2. **Pembayaran IPL + Iuran Kegiatan dalam satu form** - Satu transfer untuk keduanya
3. **File naming bukti transfer terstruktur** - Mudah dicari dan dilacak
4. **Query pembayaran by reference number** - Grouping pembayaran dalam satu transfer

## Reference Number Format

**Format:** `REF-YYYYMMDD-####`

**Contoh:**
- `REF-20250114-0001`
- `REF-20250114-0002`
- `REF-20250115-0001`

**Penjelasan:**
- `REF` - Prefix tetap
- `YYYYMMDD` - Tanggal (YYYY-MM-DD)
- `####` - Sequence number (0001-9999)

---

## File Bukti Transfer Naming

**Format Baru:** `BTF-{bulan}-{tahun}-{unit}-{timestamp}-{refNumber}.{ext}`

**Contoh:** `BTF-Jul-2026-A10-1736832000-REF-20250114-0001.jpg`

**Bagian-bagian:**
| Bagian | Contoh | Keterangan |
|--------|--------|------------|
| Prefix | `BTF` | Bukti Transfer |
| Bulan | `Jul` | Singkatan bulan Indonesia |
| Tahun | `2026` | Tahun pembayaran |
| Unit | `A10` | Nomor unit rumah |
| Timestamp | `1736832000` | Unix timestamp |
| Ref Number | `REF-20250114-0001` | Reference number (penghubung) |
| Extension | `.jpg` | File extension |

## Flow Pembayaran Baru

```
1. User buka form create IPL payment
                ↓
2. Pilih resident, periode, jumlah bulan
                ↓
3. (Optional) Input nominal iuran kegiatan
                ↓
4. Upload bukti transfer
                ↓
5. Submit
                ↓
6. Backend:
   - Generate REF-20250114-0001 (auto)
   - Create IPL payment(s) dengan ref tersebut
   - Create kegiatan payment dengan ref yang sama (jika ada)
   - Rename file bukti dengan format baru
                ↓
7. Return response dengan reference number
                ↓
8. Frontend tampilkan detail + reference number
```

## Troubleshooting

### Issue: Reference number tidak muncul di response

**Cause:** Backend belum menggunakan reference number yang di-generate

**Solution:** Pastikan backend menggunakan versi terbaru yang sudah meng-generate reference number

### Issue: File bukti transfer tidak di-rename

**Cause:** File mungkin sedang digunakan atau permission issue

**Solution:** Cek log backend untuk error saat rename file

### Issue: Iuran kegiatan tidak ter-create

**Cause:** FeeType atau Invoice creation gagal

**Solution:** Cek log backend, pastikan Resident dan HouseUnit valid
